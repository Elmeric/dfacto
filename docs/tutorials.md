## Installing dfacto

`dfacto` requires Python 3.7.2 or greater to be installed on your computer.

Install dfacto using pip or your favorite Python packages manager:

=== "Linux / macOS"
    ```shell
    $ python3 -m pip install dfacto
    ```
===+ "Windows"
    ```shell
    $ py -3 -m pip install dfacto
    ```
